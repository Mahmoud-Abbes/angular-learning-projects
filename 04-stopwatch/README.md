# 04 – Stopwatch

Angular learning project to practice timers and interval-based updates.

## Purpose

Learn how to use `setInterval`, `clearInterval`, and pipes in Angular components.

## What I practiced

- Using `setInterval` to update component state periodically
- Stopping updates with `clearInterval`
- Binding time values to the template
- Handling user events (start, stop, reset)
- Using predefined pipes

## Tech

- Angular
- TypeScript
- HTML
- CSS

> Learning project – not intended for production use.
